# tsconfig-vuepress

> TSConfig for VuePress Team

**DO NOT use it in your own project if you don't know what it's for**

## Install

```sh
npm i -D tsconfig-vuepress
```

## Usage

In `tsconfig.json`:

```json
{
  "extends": "tsconfig-vuepress/base.json"
}
```
